Справочник терминов и сокращений, используемых в документации и коде платформы RailTech B2B Ecosystem.

---

## Бизнес-термины

| Термин | Определение |
|--------|------------|
| **B2B** | Business-to-Business -- модель продаж между юридическими лицами |
| **КП** | Коммерческое предложение (Quote) -- документ с ценами и условиями для клиента |
| **ИНН** | Идентификационный номер налогоплательщика -- уникальный идентификатор компании в Узбекистане |
| **KYC** | Know Your Customer -- процедура верификации клиента (проверка ИНН, Устава) |
| **FCA** | Free Carrier -- базис поставки Инкотермс, цена на заводе-изготовителе |
| **DAP** | Delivered at Place -- базис поставки Инкотермс, доставка до Ташкента |
| **Landed Cost** | Полная себестоимость товара с учетом логистики и таможни |
| **НДС** | Налог на добавленную стоимость (12% в Узбекистане) |
| **VAT** | Value Added Tax -- англ. аналог НДС |
| **PIM** | Product Information Management -- управление информацией о товарах (мастер-каталог) |
| **Sourcing** | Процесс поиска и маппинга товаров от поставщиков |
| **Smart Match** | Автоматическое сопоставление позиций из Excel с мастер-каталогом через pg_trgm |
| **Pipeline** | Воронка продаж -- совокупность активных сделок на разных стадиях |
| **UOM** | Unit of Measure -- единица измерения (шт, кг, м, человеко-час) |
| **SKU** | Stock Keeping Unit -- уникальный артикул товара |

## Домены

| Термин | Определение |
|--------|------------|
| **RAILWAY_MACHINERY** | Домен ЖД и спецтехники -- строгий реестр с привязкой к EquipmentGroup |
| **GENERAL_GOODS** | Домен общих промышленных товаров -- плоский каталог с JSONB-атрибутами |
| **SERVICES** | Домен услуг -- упрощенное ценообразование без логистики и таможни |
| **EquipmentGroup** | Иерархический справочник техники (Тип -> Серия) для ЖД-домена |

## Роли и доступ

| Термин | Определение |
|--------|------------|
| **RBAC** | Role-Based Access Control -- ролевая модель контроля доступа |
| **GUEST** | Незарегистрированный/неверифицированный пользователь с минимальными правами |
| **CLIENT** | Верифицированный клиент с доступом к своим сделкам |
| **SALES_MANAGER** | Сейлз-менеджер RailTech, работает с воронкой и Sourcing |
| **ACCOUNT_MANAGER** | Аккаунт-менеджер RailTech, сопровождение и контрактация |
| **ADMIN** | Супер-администратор с полным доступом |
| **OWNER** | Workspace-роль: владелец компании-клиента |
| **ACCOUNTANT** | Workspace-роль: бухгалтер компании-клиента |
| **MANAGER** | Workspace-роль: снабженец компании-клиента |

## Статусы верификации компании

| Статус | Описание |
|--------|----------|
| **L1 (PROFILE_SUBMITTED)** | Профиль подан -- ожидает проверки менеджером |
| **L2 (L2_VERIFIED)** | Верифицирован -- ИНН проверен, полный доступ к платформе |
| **L3 (L3_CONTRACTED)** | Контрагент -- подписан договор, полные банковские реквизиты |

---

## Статусы сделки (Deal Stages)

| Статус | Описание |
|--------|----------|
| **DRAFT** | Черновик -- сделка создана, позиции добавляются |
| **KYC_PENDING** | Ожидание верификации документов компании |
| **CALCULATING** | Менеджер рассчитывает цены в Quote Builder |
| **QUOTE_SENT** | КП выставлено -- клиент видит Action Banner |
| **CONTRACT_SIGNED** | Договор подписан -- сделка закрыта |

## Технические термины

| **DRF** | Django REST Framework -- библиотека для построения REST API |
| **ASGI** | Asynchronous Server Gateway Interface -- протокол для async Django |

| **JSONB** | Бинарный JSON-тип данных PostgreSQL для гибких атрибутов |
| **WeasyPrint** | Python-библиотека для генерации PDF из HTML-шаблонов |
| **Zustand** | Легковесная библиотека управления состоянием для React |
| **FSM** | Finite State Machine -- конечный автомат для валидации переходов статусов |
