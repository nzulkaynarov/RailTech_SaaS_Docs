После загрузки прайс-листа ваши товары становятся доступны менеджерам RailTech для включения в коммерческие предложения клиентам. В этом разделе описано, как управлять вашими предложениями.

## Обзор кабинета поставщика

В кабинете поставщика доступны следующие разделы:

| Раздел | Описание |
|--------|----------|
| **Дашборд** | Общая статистика: количество товаров, активных запросов, сделок |
| **Мои товары** | Полный список ваших товаров на платформе |
| **Прайс-листы** | Загруженные прайс-листы и их статус |
| **Запросы** | Входящие запросы от менеджеров RailTech |
| **Статистика** | Аналитика продаж и популярности товаров |

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

## Управление товарами

### Просмотр и редактирование

1. Перейдите в раздел **«Мои товары»**
2. Используйте поиск и фильтры для нахождения нужной позиции
3. Нажмите на товар для открытия карточки

В карточке товара вы можете:

- Обновить **цену FCA**
- Изменить **наличие** (в наличии / под заказ / нет в наличии)
- Обновить **срок поставки**
- Добавить или изменить **описание и характеристики**
- Загрузить **фотографии**

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Массовое обновление цен

Для обновления цен на много товаров одновременно:

1. Перейдите в **«Мои товары»**
2. Нажмите **«Массовое обновление»**
3. Скачайте текущий список товаров в Excel
4. Обновите цены в файле
5. Загрузите файл обратно

Система обновит цены по артикулу.

### Отключение товаров

Если товар временно недоступен:

1. Откройте карточку товара
2. Переключите статус на **«Нет в наличии»**
3. Товар не будет предлагаться в новых сделках, но останется в каталоге

{{< callout type="warning" >}}
Не удаляйте товары, которые участвуют в активных сделках. Вместо этого используйте статус «Нет в наличии».
{{< /callout >}}

## Работа с запросами

Когда менеджер RailTech ищет товар для клиента, вы можете получить запрос.

### Типы запросов

- **Запрос цены** — менеджер уточняет актуальность цены из прайс-листа
- **Запрос наличия** — подтверждение наличия и сроков поставки
- **Специальный запрос** — товар, которого нет в вашем прайс-листе

### Ответ на запрос

1. Откройте раздел **«Запросы»**
2. Найдите запрос со статусом **«Новый»**
3. Заполните ответ:
   - Подтвердите или обновите цену
   - Укажите актуальный срок поставки
   - Добавьте комментарий при необходимости
4. Нажмите **«Отправить ответ»**

{{< callout type="info" >}}
Рекомендуемый срок ответа на запрос — **1 рабочий день**. Оперативность влияет на приоритет ваших товаров при формировании КП.
{{< /callout >}}

## Аналитика

В разделе **«Статистика»** доступна информация:

- Количество просмотров ваших товаров
- Количество включений в КП
- Топ запрашиваемых товаров
- Динамика запросов по месяцам

Используйте аналитику для оптимизации ассортимента и ценообразования.

## Советы по повышению продаж

- **Поддерживайте актуальные цены** — устаревшие цены замедляют обработку сделок
- **Добавляйте фотографии** — товары с фото получают больше внимания
- **Указывайте точные сроки** — клиенты ценят предсказуемость
- **Отвечайте на запросы быстро** — это повышает ваш рейтинг на платформе
- **Расширяйте ассортимент** — добавляйте сопутствующие товары
