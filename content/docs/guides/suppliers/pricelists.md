Прайс-лист — основа работы на платформе. Актуальные цены позволяют менеджерам RailTech быстро формировать коммерческие предложения для клиентов.

## Формат прайс-листа

### Поддерживаемые форматы файлов

- **XLSX** — рекомендуется
- **XLS** — поддерживается
- **CSV** — для простых списков

### Обязательные колонки

| Колонка | Описание | Пример |
|---------|----------|--------|
| Артикул | Уникальный код товара | KP-100M |
| Наименование | Полное название товара | Колёсная пара КП-100М |
| Цена FCA | Цена на условиях FCA | 15 200.00 |
| Валюта | USD, EUR или UZS | USD |
| Ед. изм. | Единица измерения | шт |
| Наличие | Есть на складе / под заказ | В наличии |
| Срок поставки | Дней до отгрузки | 14 |

### Дополнительные колонки (рекомендуемые)

| Колонка | Описание |
|---------|----------|
| Категория | Категория товара |
| Производитель | Бренд / завод-изготовитель |
| Страна | Страна происхождения |
| Мин. партия | Минимальный объём заказа |
| Вес | Вес единицы (кг) |
| Описание | Краткое техническое описание |

## Загрузка прайс-листа

### Шаг 1. Откройте раздел «Прайс-листы»

В кабинете поставщика перейдите в раздел **«Мои прайс-листы»** в боковом меню.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 2. Нажмите «Загрузить прайс-лист»

Нажмите кнопку **«+ Загрузить»** в правом верхнем углу.

### Шаг 3. Выберите файл и укажите параметры

1. Перетащите файл или нажмите **«Выбрать файл»**
2. Укажите **название прайс-листа** (например: «Основной прайс Q1 2026»)
3. Выберите **валюту по умолчанию**
4. Укажите **дату действия** (с какого по какое число)

### Шаг 4. Сопоставьте колонки

Система автоматически попробует распознать колонки. Проверьте и при необходимости скорректируйте сопоставление.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 5. Просмотрите и подтвердите

Система покажет предварительный просмотр загруженных данных:

- Количество распознанных позиций
- Ошибки валидации (если есть)
- Позиции с предупреждениями

Исправьте ошибки и нажмите **«Подтвердить загрузку»**.

{{< callout type="warning" >}}
Если в файле есть строки без артикула или цены, они будут пропущены. Проверьте раздел «Пропущенные строки» после загрузки.
{{< /callout >}}

## Обновление прайс-листа

Для обновления цен:

1. Откройте существующий прайс-лист
2. Нажмите **«Обновить»**
3. Загрузите новый файл
4. Система автоматически сопоставит позиции по артикулу и обновит цены

Позиции, которых нет в новом файле, останутся без изменений. Новые позиции будут добавлены.

## Рекомендации

- **Обновляйте цены регулярно** — минимум 1 раз в месяц
- **Используйте единый формат артикулов** — это упрощает сопоставление
- **Указывайте цены FCA** — логистику и таможню рассчитывает RailTech
- **Не забывайте сроки поставки** — это важный параметр для клиентов

{{< callout type="info" >}}
Вы можете скачать шаблон прайс-листа в разделе «Прайс-листы» вашего кабинета.
{{< /callout >}}
