Every deal on the platform has a built-in chat for real-time communication with your manager. The chat works in real time and preserves the full message history.

## Where to Find the Chat

The chat is available on every deal page. Open a deal and go to the **"Chat"** tab or click the chat icon on the right side of the screen.

## Chat Features

### Text Messages

Type your message in the input field and press **Enter** or click the **"Send"** button. The manager will be notified instantly.

### File Attachments

You can attach files to a message:

1. Click the **paperclip icon** next to the input field
2. Select a file (up to 25 MB)
3. Supported formats: PDF, XLSX, DOCX, JPG, PNG

This is convenient for sharing:

- Technical drawings
- Additional specifications
- Equipment photos
- Document scans

### Presence Indicator

A green dot next to the manager's name means they are online and can respond immediately. A gray dot means the manager is offline but will receive a notification.

## Message Notifications

You will receive a notification about a new message:

- **On the platform** -- the bell icon will show the number of unread messages
- **Via email** -- if you have not read the message within 15 minutes
- **Sound alert** -- when a message is received while the tab is active

{{< callout type="info" >}}
Notification settings can be changed in **"Settings" > "Notifications"**.
{{< /callout >}}

## Deal Context in Chat

The top of the chat displays brief information about the deal:

- Deal name and number
- Current stage
- Number of items

This helps you quickly orient yourself if you have multiple active deals.

## Usage Tips

- **Ask specific questions** -- the manager will respond faster
- **Attach documents** via chat rather than email -- all correspondence will be in one place
- **Use the chat for clarifications** on commercial quotes
- **The chat is per-deal** -- each deal has a separate chat

{{< callout type="warning" >}}
Do not share confidential information (passwords, payment card details) through the chat. Use the KYC section for exchanging bank details.
{{< /callout >}}

## Support Working Hours

Managers are available in chat during business hours: **Mon--Fri, 09:00--18:00 (Tashkent, UTC+5)**. Messages sent outside working hours will be processed at the start of the next business day.
