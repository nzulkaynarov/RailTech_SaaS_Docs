Заявка (сделка) — это ваш запрос на закупку оборудования. После создания заявки менеджер RailTech подберёт лучшие предложения и подготовит коммерческое предложение.

{{< callout type="info" >}}
Для создания заявок необходимо пройти [KYC-верификацию](../kyc/).
{{< /callout >}}

## Способы создания заявки

Существует два способа:

1. **Вручную** — добавляйте позиции из каталога по одной
2. **Импорт из Excel** — загрузите готовую спецификацию (см. [Импорт Excel](../excel-import/))

## Создание заявки вручную

### Шаг 1. Перейдите в раздел «Мои сделки»

В личном кабинете откройте раздел **«Сделки»** в боковом меню.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 2. Нажмите «Создать заявку»

Нажмите кнопку **«+ Новая заявка»** в правом верхнем углу.

### Шаг 3. Заполните основную информацию

| Поле | Описание | Обязательное |
|------|----------|:------------:|
| Название заявки | Краткое описание потребности | Да |
| Приоритет | Стандартный / Срочный | Нет |
| Комментарий | Дополнительные пожелания | Нет |
| Желаемая дата поставки | Когда нужно оборудование | Нет |

### Шаг 4. Добавьте позиции

Нажмите **«Добавить позицию»** и выберите один из вариантов:

- **Из каталога** — найдите товар через поиск и укажите количество
- **Произвольная позиция** — опишите товар текстом, если не нашли в каталоге

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

Для каждой позиции укажите:

- Наименование или выберите из каталога
- Количество
- Единицу измерения
- Комментарий (опционально) — например, «аналог допустим»

### Шаг 5. Отправьте заявку

Нажмите **«Отправить заявку»**. Сделка перейдёт в обработку.

## Стадии сделки

После отправки заявка проходит несколько стадий:

| Стадия | Что происходит |
|--------|----------------|
| **Черновик** | Вы формируете заявку, ещё не отправлена |
| **KYC ожидание** | Если KYC ещё не пройден — сделка ждёт верификации |
| **Подбор (Sourcing)** | Менеджер подбирает поставщиков и цены |
| **Расчёт** | Менеджер рассчитывает стоимость доставки и итоговую цену |
| **КП отправлено** | Вам направлено коммерческое предложение |
| **Договор подписан** | Вы согласовали КП и подписали договор |
| **Закрыта (выиграна)** | Сделка успешно завершена |

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

## Отслеживание статуса

В разделе **«Мои сделки»** вы видите список всех ваших заявок с текущим статусом. Нажмите на заявку для просмотра деталей.

На странице сделки доступно:

- **Список позиций** — что вы запросили
- **Текущая стадия** — на каком этапе обработка
- **Коммерческое предложение** — когда менеджер его подготовит
- **Чат с менеджером** — для обсуждения деталей
- **История изменений** — все действия по сделке

## Редактирование заявки

Вы можете редактировать заявку, пока она находится в стадии **«Черновик»**:

- Добавлять и удалять позиции
- Менять количество
- Обновлять комментарии

{{< callout type="warning" >}}
После отправки заявки изменения вносятся только через менеджера. Напишите ему в [чат сделки](../chat/).
{{< /callout >}}

## Советы

- Указывайте максимум деталей: артикулы, чертежи, допустимые аналоги
- Для больших спецификаций (10+ позиций) используйте [импорт из Excel](../excel-import/)
- Если не нашли товар в каталоге — опишите его текстом, менеджер подберёт
