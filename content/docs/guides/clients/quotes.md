Когда менеджер RailTech рассчитает стоимость по вашей заявке, вы получите коммерческое предложение (КП). В этом руководстве описано, как работать с КП на платформе.

## Как вы получите КП

1. Менеджер обрабатывает вашу заявку и подбирает поставщиков
2. Рассчитывается стоимость с учётом логистики, таможни и наценки
3. Вы получаете **уведомление на email** и **в личном кабинете**
4. Стадия сделки меняется на **«КП отправлено»**

## Просмотр коммерческого предложения

### Шаг 1. Откройте сделку

Перейдите в **«Сделки»** и откройте сделку со статусом **«КП отправлено»**.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 2. Перейдите на вкладку «Коммерческое предложение»

На странице сделки откройте вкладку **«КП»** или **«Коммерческое предложение»**.

### Шаг 3. Изучите предложение

КП содержит таблицу с расчётом по каждой позиции:

| Колонка | Описание |
|---------|----------|
| Наименование | Предложенный товар |
| Количество | Запрошенное количество |
| Цена за единицу | Стоимость единицы (включая доставку) |
| Сумма | Итого по позиции |
| НДС (12%) | Сумма НДС |
| Итого с НДС | Финальная стоимость |
| Срок поставки | Ожидаемый срок |
| Примечание | Комментарий менеджера |

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

В нижней части таблицы отображается:

- **Итого без НДС** — сумма по всем позициям
- **НДС 12%** — общая сумма налога
- **Итого с НДС** — финальная сумма к оплате

{{< callout type="info" >}}
Все цены указаны в узбекских сумах (UZS). Курсы пересчёта валют фиксируются на дату формирования КП.
{{< /callout >}}

## Действия с предложением

### Согласовать КП

Если предложение вас устраивает:

1. Нажмите **«Согласовать»**
2. Подтвердите действие во всплывающем окне
3. Сделка перейдёт в стадию **«Договор подписан»**
4. Менеджер свяжется с вами для оформления договора

### Запросить изменения

Если нужны корректировки:

1. Нажмите **«Запросить изменения»**
2. Опишите, что именно нужно изменить (цена, количество, замена позиции)
3. Менеджер получит уведомление и подготовит обновлённое КП

Вы также можете обсудить детали через [чат сделки](../chat/).

### Отклонить КП

Если предложение вам не подходит:

1. Нажмите **«Отклонить»**
2. Укажите причину отклонения
3. Сделка будет закрыта, но вы сможете создать новую заявку позже

{{< callout type="warning" >}}
Отклонение КП не является окончательным. Вы всегда можете создать новую заявку с обновлёнными требованиями.
{{< /callout >}}

## Скачивание КП

Вы можете скачать КП в формате PDF для внутреннего согласования:

1. Нажмите **«Скачать PDF»** на странице КП
2. Файл будет сохранён на ваш компьютер

PDF-версия содержит все данные предложения, реквизиты сторон и QR-код для быстрого доступа к сделке на платформе.

## Срок действия КП

Коммерческое предложение действует **14 дней** с момента отправки (если менеджер не указал иной срок). По истечении срока цены могут быть пересчитаны.
