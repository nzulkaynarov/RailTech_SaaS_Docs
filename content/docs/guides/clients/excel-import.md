Если у вас есть готовая спецификация в формате Excel, вы можете загрузить её на платформу. Система автоматически распознает позиции и сопоставит их с товарами каталога.

## Поддерживаемые форматы

- **XLSX** — Microsoft Excel 2007 и новее (рекомендуется)
- **XLS** — Microsoft Excel 97–2003
- Максимальный размер файла: **15 МБ**
- Максимальное количество строк: **500**

## Требования к файлу

Ваш Excel-файл должен содержать как минимум:

| Колонка | Описание | Обязательно |
|---------|----------|:-----------:|
| Наименование | Название товара / запчасти | Да |
| Количество | Нужное количество | Да |
| Ед. изм. | Единица измерения (шт, кг, м) | Желательно |
| Артикул | Каталожный номер | Желательно |
| Комментарий | Примечания к позиции | Нет |

{{< callout type="info" >}}
Система гибко распознаёт заголовки колонок. Например, «Кол-во», «Количество», «Qty» — будут распознаны как количество.
{{< /callout >}}

## Пошаговая инструкция

### Шаг 1. Создайте новую заявку или откройте существующую

Перейдите в раздел **«Сделки»** и создайте новую заявку или откройте черновик.

### Шаг 2. Нажмите «Импорт из Excel»

На странице сделки нажмите кнопку **«Импорт из Excel»** в блоке позиций.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 3. Загрузите файл

Перетащите файл в область загрузки или нажмите **«Выбрать файл»**. Дождитесь завершения обработки.

### Шаг 4. Проверьте распознанные данные

После загрузки система покажет предварительный просмотр:

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

Для каждой строки будет показано:

- **Распознанное наименование** — что система прочитала из файла
- **Сопоставленный товар** — ближайший товар в каталоге (если найден)
- **Степень совпадения** — насколько точно найден товар (в процентах)
- **Количество и единица измерения**

### Шаг 5. Подтвердите или скорректируйте

Для каждой позиции вы можете:

- **Принять** автоматическое сопоставление, если оно верное
- **Изменить** — выбрать другой товар из каталога вручную
- **Оставить как есть** — если товар не найден, менеджер подберёт его вручную

### Шаг 6. Сохраните результат

Нажмите **«Подтвердить импорт»**. Позиции будут добавлены в вашу заявку.

## Как работает автоматическое сопоставление

Система использует технологию **Smart Match** для поиска соответствий:

1. Наименование из вашего файла сравнивается с каталогом по нечёткому совпадению
2. Учитываются артикулы, синонимы и альтернативные названия
3. Результат ранжируется по степени схожести

Позиции, которые не удалось сопоставить автоматически, попадают на ручную обработку менеджером.

{{< callout type="warning" >}}
Чем точнее вы указываете артикулы и наименования в спецификации, тем лучше работает автоматическое сопоставление.
{{< /callout >}}

## Советы по подготовке файла

- Используйте **один лист** (sheet) в файле — система читает первый лист
- Поместите **заголовки в первую строку**
- Не объединяйте ячейки
- Избегайте пустых строк между данными
- Указывайте артикулы — это значительно повышает точность сопоставления
