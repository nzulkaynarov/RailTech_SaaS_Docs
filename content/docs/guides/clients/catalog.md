Каталог RailTech B2B содержит тысячи наименований железнодорожного оборудования, запасных частей и промышленных товаров. В этом руководстве вы узнаете, как эффективно находить нужные позиции.

## Структура каталога

Каталог разделён на три домена:

| Домен | Описание | Пример |
|-------|----------|--------|
| **Железнодорожное оборудование** | Подвижной состав, запчасти, узлы | Колёсная пара, тележка, редуктор |
| **Общие товары** | Промышленные товары и расходные материалы | Подшипники, смазочные материалы, инструмент |
| **Услуги** | Сервисные услуги | Диагностика, ремонт, сертификация |

Железнодорожное оборудование имеет строгую иерархию категорий и привязку к группам оборудования. Общие товары — свободную структуру с гибкими атрибутами.

## Поиск товаров

### Быстрый поиск

Воспользуйтесь строкой поиска в верхней части каталога. Введите название, артикул или ключевые слова.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

**Советы по поиску:**

- Используйте точный артикул для быстрого нахождения: `КП-100М`
- Ищите по части названия: `колёсная пара`
- Система поддерживает поиск по техническим характеристикам

### Фильтрация по категориям

Слева от списка товаров расположено дерево категорий. Нажмите на категорию, чтобы увидеть товары только из неё.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

Для железнодорожного оборудования доступна дополнительная фильтрация по **группе оборудования** (например: «Локомотивы», «Вагоны», «Путевая техника»).

### Дополнительные фильтры

В зависимости от категории доступны фильтры:

- **Производитель** — выберите одного или нескольких
- **Страна происхождения** — фильтр по стране
- **Наличие** — только товары в наличии
- **Технические параметры** — зависят от категории (диаметр, мощность, вес и т.д.)

## Карточка товара

Нажмите на товар в списке, чтобы открыть полную карточку.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

Карточка содержит:

- **Основная информация** — название, артикул, описание
- **Технические характеристики** — таблица параметров
- **Фотографии** — галерея изображений товара
- **Категория и группа оборудования** — место в иерархии каталога
- **Цена** — отображается только для верифицированных клиентов

{{< callout type="warning" >}}
Если вместо цены вы видите надпись «Цена по запросу» — необходимо пройти [KYC-верификацию](../kyc/).
{{< /callout >}}

## Добавление в заявку

Из карточки товара вы можете сразу добавить позицию в заявку:

1. Укажите **количество**
2. Нажмите **«Добавить в заявку»**
3. Выберите существующую сделку или создайте новую

Подробнее о создании сделок — в разделе [Создание заявки](../deals/).

## Единицы измерения

В каталоге используются стандартные единицы измерения:

- **шт** — штуки (для дискретных товаров)
- **компл** — комплекты
- **кг**, **т** — килограммы, тонны (для весовых товаров)
- **м**, **п.м.** — метры, погонные метры
- **л** — литры (для жидкостей)
