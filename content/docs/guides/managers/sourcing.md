Sourcing Hub — центр обработки позиций, которые Smart Match не смог автоматически сопоставить с каталогом. Здесь менеджер вручную находит товары, создаёт новые карточки или отклоняет нерелевантные запросы.

## Как позиции попадают в Sourcing Hub

Когда клиент загружает Excel-спецификацию, Smart Match пытается сопоставить каждую строку с каталогом. Позиции с коэффициентом схожести **менее 50%** автоматически попадают в Sourcing Hub как **UnrecognizedRow** (нераспознанная строка).

Типичные причины:

- Товар отсутствует в каталоге
- Клиент использует нестандартное наименование
- Опечатки или сокращения в спецификации
- Товар из другой отрасли (не типичный для каталога)

## Интерфейс Sourcing Hub

### Список нераспознанных позиций

Откройте раздел **«Sourcing Hub»** в боковом меню CRM.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

Для каждой позиции отображается:

| Колонка | Описание |
|---------|----------|
| **Исходный текст** | Что написал клиент в Excel |
| **Сделка** | К какой сделке относится позиция |
| **Клиент** | Компания-заказчик |
| **Дата** | Когда загружена спецификация |
| **Статус** | Новая / В обработке / Сопоставлена / Отклонена |
| **Ближайшие совпадения** | Лучшие варианты из каталога (если есть) |

### Фильтры

Используйте фильтры для удобной работы:

- **По статусу** — показать только новые
- **По сделке** — позиции конкретной сделки
- **По дате** — за период
- **По клиенту** — позиции от конкретного клиента

## Обработка нераспознанных позиций

### Вариант 1: Сопоставить с существующим товаром

Если товар есть в каталоге, но не был распознан:

1. Откройте нераспознанную позицию
2. В поле **«Поиск по каталогу»** введите название или артикул
3. Выберите правильный товар из результатов
4. Нажмите **«Сопоставить»**

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

{{< callout type="info" >}}
При сопоставлении система запоминает связку «исходный текст → товар каталога». В будущем аналогичные позиции будут распознаваться автоматически.
{{< /callout >}}

### Вариант 2: Создать новую карточку товара

Если товар отсутствует в каталоге:

1. Нажмите **«Создать товар»**
2. Заполните карточку нового товара:
   - Наименование
   - Артикул
   - Категория (выберите из иерархии)
   - Домен (Железнодорожное оборудование / Общие товары / Услуги)
   - Группа оборудования (для ж/д оборудования — обязательно)
   - Технические характеристики
3. Сохраните карточку
4. Сопоставьте нераспознанную позицию с новым товаром

{{< callout type="warning" >}}
При создании товара в домене «Железнодорожное оборудование» обязательно укажите группу оборудования (EquipmentGroup). Без неё сохранение невозможно.
{{< /callout >}}

### Вариант 3: Отклонить позицию

Если позиция нерелевантна (ошибка клиента, дубль, не товар):

1. Нажмите **«Отклонить»**
2. Укажите причину отклонения
3. Клиент и менеджер сделки увидят комментарий

## Массовая обработка

Для ускорения работы с большими спецификациями:

1. Отметьте несколько позиций галочками
2. Нажмите **«Массовое действие»**
3. Выберите действие:
   - **Сопоставить с одним товаром** — если несколько строк относятся к одному товару
   - **Отклонить выбранные** — для нерелевантных позиций

## Правила работы с доменами

При сопоставлении или создании товаров соблюдайте доменные правила:

| Домен | Группа оборудования | JSONB-атрибуты | Строгая иерархия |
|-------|:-------------------:|:--------------:|:----------------:|
| Железнодорожное оборудование | Обязательна | Частично | Да |
| Общие товары | Нет | Да | Нет |
| Услуги | Нет | Нет | Нет |

## Отчёты

В разделе отчётов Sourcing Hub доступно:

- **Топ нераспознанных позиций** — какие товары чаще всего не распознаются
- **Статистика обработки** — сколько позиций обработано за период
- **Эффективность Smart Match** — процент автоматических совпадений

Используйте эти данные для пополнения каталога: если одна и та же позиция часто не распознаётся — добавьте её в каталог.

## Советы

- Обрабатывайте Sourcing Hub ежедневно — не допускайте накопления
- При создании товаров используйте стандартные наименования из ГОСТ или ТУ
- Добавляйте синонимы к карточке товара после сопоставления
- Консультируйтесь с отделом закупок при сомнениях в классификации
