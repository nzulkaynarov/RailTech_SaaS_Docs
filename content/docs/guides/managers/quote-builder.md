Quote Builder — инструмент для расчёта итоговой стоимости товаров с учётом логистики, таможни, наценки и НДС. Результат — готовое коммерческое предложение для клиента.

## Формула расчёта

Для товаров (железнодорожное оборудование и общие товары):

```
Цена FCA (базовая цена поставщика)
  + Логистика DAP (доставка до Узбекистана)
  + Таможенная пошлина
  = Landed Cost (себестоимость доставленного товара)
    × (1 + Маржа %)
    = Цена без НДС
      × 1.12
      = Итоговая цена с НДС (12%)
```

Для услуг (упрощённая формула):

```
Базовая ставка × (1 + Маржа %) × 1.12 = Итоговая цена с НДС
```

{{< callout type="warning" >}}
Все финансовые расчёты выполняются на сервере. Никогда не рассчитывайте итоговые цены вручную — используйте Quote Builder.
{{< /callout >}}

## Работа с Quote Builder

### Шаг 1. Откройте Quote Builder

В карточке сделки на стадии **CALCULATING** перейдите на вкладку **«Quote Builder»** или нажмите кнопку **«Рассчитать стоимость»**.

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 2. Проверьте базовые цены

Для каждой позиции отображается:

| Поле | Описание |
|------|----------|
| Товар | Наименование и артикул |
| Поставщик | Выбранный поставщик |
| Цена FCA | Базовая цена от поставщика |
| Валюта | Валюта поставщика |
| Курс | Текущий курс конвертации |
| Кол-во | Количество единиц |

Убедитесь, что для каждой позиции выбран поставщик и указана актуальная цена.

### Шаг 3. Укажите параметры логистики

Заполните параметры доставки:

| Параметр | Описание |
|----------|----------|
| **Стоимость доставки** | Общая стоимость транспортировки (можно на всю партию или на позицию) |
| **Условия доставки** | DAP (доставка до места назначения) |
| **Способ доставки** | Ж/д, авто, морской, мультимодальный |
| **Страна отправления** | Откуда доставляется товар |

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 4. Укажите таможенные расходы

| Параметр | Описание |
|----------|----------|
| **Ставка пошлины** | Процент от стоимости товара (зависит от кода ТН ВЭД) |
| **Таможенный сбор** | Фиксированный сбор за оформление |
| **Сертификация** | Расходы на сертификацию (при необходимости) |

### Шаг 5. Установите маржу

Укажите процент наценки RailTech:

- **Стандартная маржа** — применяется ко всем позициям
- **Индивидуальная маржа** — можно установить для каждой позиции отдельно

{{< callout type="info" >}}
Рекомендуемый диапазон маржи зависит от категории товара и объёма сделки. Обратитесь к руководителю отдела продаж для согласования.
{{< /callout >}}

### Шаг 6. Просмотрите расчёт

Quote Builder автоматически рассчитает и покажет:

| Показатель | Формула |
|------------|---------|
| **Landed Cost** | FCA + Логистика + Таможня |
| **Цена продажи (без НДС)** | Landed Cost × (1 + Маржа%) |
| **НДС 12%** | Цена продажи × 0.12 |
| **Итого с НДС** | Цена продажи × 1.12 |
| **Маржа (сумма)** | Цена продажи - Landed Cost |
| **Маржинальность %** | Маржа / Цена продажи × 100 |

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 7. Сформируйте КП

Нажмите **«Сформировать КП»**. Система создаст документ коммерческого предложения.

Перед отправкой клиенту вы можете:

- Просмотреть КП в режиме предпросмотра
- Скачать PDF для внутреннего согласования
- Добавить комментарий для клиента
- Указать срок действия предложения

### Шаг 8. Отправьте КП клиенту

Нажмите **«Отправить клиенту»**. Сделка перейдёт в стадию **QUOTE_SENT**.

## Валюта и курсы

- Базовые цены поставщиков могут быть в USD, EUR или UZS
- Система автоматически конвертирует в UZS по текущему курсу ЦБ
- Курс фиксируется на дату формирования КП
- Для пересчёта по новому курсу нажмите **«Обновить курсы»**

## Важные правила

- Используйте только `DecimalField` — никогда не округляйте вручную
- Проверяйте курс валюты перед формированием КП
- Согласовывайте маржу на крупные сделки с руководством
- Все расчёты сохраняются в истории сделки для аудита
