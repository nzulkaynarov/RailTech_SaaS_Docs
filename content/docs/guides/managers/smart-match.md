Smart Match — ключевая функция платформы, которая автоматически сопоставляет позиции из клиентских Excel-спецификаций с товарами каталога. Это значительно ускоряет обработку заявок.

## Как работает Smart Match

Алгоритм использует технологию **pg_trgm** (триграммное сходство) для нечёткого сопоставления текстов:

1. Система извлекает наименование и артикул из каждой строки Excel
2. Выполняется поиск по каталогу с использованием триграммного индекса
3. Для каждой позиции возвращается список кандидатов с **коэффициентом схожести** (0–100%)
4. Позиции с высоким коэффициентом автоматически сопоставляются
5. Позиции с низким коэффициентом попадают в [Sourcing Hub](../sourcing/) для ручной обработки

### Пороги совпадения

| Коэффициент | Статус | Действие |
|:-----------:|--------|----------|
| **≥ 80%** | Автоматическое совпадение | Позиция сопоставлена, менеджер подтверждает |
| **50–79%** | Предложения | Система предлагает варианты, менеджер выбирает |
| **< 50%** | Не распознано | Позиция попадает в Sourcing Hub |

## Обработка импортированных файлов

### Шаг 1. Откройте сделку

Перейдите в карточку сделки, где клиент загрузил Excel-спецификацию.

На вкладке **«Позиции»** вы увидите результат автоматического сопоставления.

### Шаг 2. Проверьте автоматические совпадения

Для каждой позиции отображается:

- **Исходный текст** — что написал клиент
- **Сопоставленный товар** — что нашла система
- **Коэффициент** — степень уверенности
- **Статус** — автоматическое/предложение/не распознано

Цветовая индикация:

- **Зелёный** — автоматическое совпадение (≥ 80%)
- **Жёлтый** — есть предложения (50–79%)
-  **Красный** — не распознано (< 50%)

### Шаг 3. Подтвердите или измените совпадения

Для каждой позиции:

- **Подтвердите** автоматическое совпадение — нажмите галочку
- **Выберите из предложений** — нажмите на альтернативный вариант
- **Найдите вручную** — используйте поиск по каталогу
- **Отправьте в Sourcing** — если товар не найден в каталоге

{{< callout type="info" >}}Раздел в разработке. Скриншоты будут добавлены в следующем обновлении.{{< /callout >}}

### Шаг 4. Обработайте нераспознанные позиции

Позиции, которые не удалось сопоставить, автоматически попадают в **Sourcing Hub**. Подробнее — в разделе [Sourcing Hub](../sourcing/).

## Массовое подтверждение

Если в спецификации много позиций с высоким коэффициентом совпадения:

1. Нажмите **«Выбрать все автоматические»** — отмечаются все позиции с коэффициентом ≥ 80%
2. Проверьте список визуально
3. Нажмите **«Подтвердить выбранные»**

{{< callout type="warning" >}}
Всегда проверяйте автоматические совпадения перед массовым подтверждением. Алгоритм может ошибаться, особенно для товаров с похожими названиями, но разными характеристиками.
{{< /callout >}}

## Повышение качества Smart Match

Качество сопоставления зависит от полноты каталога:

- **Добавляйте синонимы** к товарам в каталоге
- **Указывайте альтернативные артикулы** — OEM, оригинальные, заводские
- **Заполняйте все поля** карточки товара
- **Обновляйте каталог** при появлении новых наименований от клиентов

## Статистика Smart Match

В разделе отчётов доступна статистика:

- Общий процент автоматических совпадений
- Среднее время обработки спецификации
- Топ нераспознанных позиций (кандидаты для добавления в каталог)
