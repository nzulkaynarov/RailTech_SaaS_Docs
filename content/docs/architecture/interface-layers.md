## Слой 1: Landing Page (`railtech.uz`)
- **Тип:** Статический маркетинговый сайт.
- **Задача:** SEO-трафик, презентация компании как единого поставщика для промышленности (Оборудование, Материалы, Услуги), переход в B2B-портал.

## Слой 2: B2B Каталог (`b2b.railtech.uz`)
- **Тип:** Публичное SPA (React). Цены скрыты.
- **Интерфейсы:**
  - *Фасетный поиск (Domain-Adaptive):* Боковая панель с фильтрами динамически меняется в зависимости от выбранного Домена. Для ЖД появляются строгие фильтры по `EquipmentGroup` (Тепловозы/Электровозы/Вагоны). Для общих товаров -- фильтры по категориям и атрибутам.
  - *Мультивидовость:* Переключение между «Компактной сеткой» и «Таблицей» (Data Grid). Пустые состояния каталога явно подсказывают о возможности загрузки Excel-спецификаций.
  - *Quick Order:* Быстрое добавление товаров и услуг в корзину через кнопку "В запрос".

## Слой 3: Личный кабинет клиента (`cabinet.railtech.uz`)
- **Тип:** Закрытое SPA (React).
- **Интерфейсы:**
  - *Дашборд:* Интерактивные KPI-виджеты (выступают фильтрами для таблицы истории).
  - *Навигация:* Сворачиваемый Sidebar (Collapsible) для расширения рабочей зоны. Для профилей L1 отображается яркий прогресс-бар и призыв загрузить Устав компании.
  - *Deal View (Детализация):* Сплит-экран (70/30). Слева -- спецификация с визуальной группировкой позиций по Доменам. В пустом состоянии предлагаются крупные CTA "Импорт Excel" и "В Каталог". Если статус сделки `QUOTE_SENT`, клиент видит Action Banner с кнопкой "Принять КП". Справа -- контекстный чат (WebSockets) и зона PDF-документов.

## Слой 4: Админ-панель / CRM
- **Тип:** Защищенное SPA (React + Tailwind CSS + Lucide React) / Кастомный Django Admin (Unfold).
- **Интерфейсы:**
  - *Сайдбар:* Темная тема, раскрывается при наведении (Hover-expand).
  - *KYC Pane:* Карточка контрагента с кнопкой проверки по базам.
  - *Quote Builder:* Сложная таблица с расчетом юнит-экономики (Закупка -> Накладные -> С/С -> Цена). Визуальное разделение позиций ЖД-техники, общих товаров и услуг. Внутренняя маржа скрыта от внешних экспортов.
  - *Sourcing Hub:* Модуль для ручного маппинга импортированных из Excel, но нераспознанных позиций (`product=null`).
  - *Коммерческая Аналитика:* Дашборды с Conversion Rate, Pipeline Volume, и статусами сделок.
